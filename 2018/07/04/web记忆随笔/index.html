web记忆随笔-html&css

2018-07-04
web

html篇

  • 像h1, p, ul, li, a, dl, dt, dd, img中的alt等这些语义化的标签能方便搜索引擎收录.

  • a中的title属性，鼠标放上去会有提示.

  • <dl><dt></dt><dd></dd></dl> -> emmet: dl>(dt+dd)*3

  • <ul><li></li><li></li></ul> -> emmet: ul>li*3

  • 简历表格可用table标签完成.

  • input的radio和checkbox类型，需要name相同来实现单选和复选.

  • <label for="username">名字</label><input type="text" id="username">
    label里面加for，能实现点击文字也能激活input. checkbox同理，用label包围文字.

  • input的hidden类型，可以用来存储值.

  • input的image类型不要用，有bug.

  • iframe中定义name属性，在a中的target属性写上name，能使iframe加载a中的链接.

css篇

  • 首页使用内嵌样式（在head中写style），能提升首页加载性能，见淘宝.

  • 用img可以让搜索引擎爬取，background-image不行.

  • padding:10px(上) 20px(左右) 30px(下);

  • padding变长，border也会变长. 解决办法：用text-indent:15px;来替代padding-left:15px;

  • 块元素水平居中：margin:100px auto;

  • 上下margin的合并(左右margin没有合并)，最终等于两者中较大的那个.只有块元素有margin合并问题，行内块和内联元素不会合并.

  • body默认有8px的margin.

  • margin-top(其他方向没有)的塌陷问题：父div包裹设置了margin-top的子div，父div也会产生margin-top. 解决办法： 1.父div设置边框(基本不用). 2.父div设置overflow:hidden;(这个子div如果溢出，会被裁减). 3.父div加入:before{content:"";display:table;}.（类似加了边框）

  • 块元素

    • 支持所有样式

    • 如果没有设置宽度，默认宽度为父级的100%.

    • 占据一行，即使设置了宽度

  • 行内元素

    • 支持部分样式（不支持宽高、margin上下、padding上下）

    • 宽高由内容决定.

    • 两个<a>标签之间代码换行的话，两个<a>之间会产生间距，解决办法：将父元素的font-size设置为0，然后将<a>的font-size设置为目标大小.

    • 父元素如果是块级元素，父元素使用text-align能够使子元素水平居中，使用line-height使子元素垂直居中.

  • 内联块元素（img和input是特殊的）

    • 支持全部样式.

    • 如果没设置宽高，由内容决定.

    • 两个<a>标签即使设置成inline-block，他们之间代码换行的话，还是会有间距问题.

    • 父元素如果是块级元素，父元素使用text-align能够使子元素水平居中，使用line-height使子元素垂直居中.

  • 浮动

    • 文字环绕通过浮动完成，其中div1浮动，div2不浮动，div2的文字会自动环绕div1.

    • 清除浮动

      • 1.父级元素overflow:hidden;;
      • 2.在最后一个子元素后面加一个空的div，然后clear:both;;

      • 3..clearfix:before, .clearfix:after{content:''; display:table; }，.clearfix:after{clear:both}，还要加个.clearfix{zoom:1}这个是用来兼容IE的，因为有些版本不支持:after.

        ps:上面第3点是最完整的清除margin-top塌陷和清除浮动的代码.

  • 定位

    • relative : 相对于自身元素偏移，元素依然占位原来的位置;

    • absolute : 相对于设置了fixed,relative,absolute的父元素进行绝对定位，如果都没有，则相对于body.绝对定位的元素的margin是有效的.

    • fixed : fixed是相对于浏览器窗口进行定位.

      ps:绝对定位和固定定位的块元素和内联元素会转换成行内块元素
