#redis
数据库

Redis

Redis和MongoDB一样，属于NoSQL变种，他们都是基于内存的读写，当然，最终还是依赖于磁盘存储，Redis基于key-value存储，MongoDB基于json存储.

安装配置

mac直接通过：brew install redis安装

启动：redis-server /usr/local/etc/redis.conf(默认端口为6379)

客户端连接：redis-cli

可视化客户端软件：Redis Destop Manager

常用命令

select 1 redis默认有16个数据库，这里表示使用第1个

keys * 查看所有key

set key value 存储string

get key 读取

setex key seconds value 设置有效时间

type key 类型：string、hash、list、set、zset

hset key attr_name attr_value 存储hash对象

hget key attr_name 读取hash对象

lpush key attr_name attr_value 存储list

lpop key 读取list

sadd key value1 value2 valu3 ... 存储set

smembers key 读取set

zadd key score1 value1 score2 value2 ... zset是有序的set集合，score表示顺序

zrange key 0 -1 查看从头到尾的zset集合

主从配置

在主redis服务器redis.conf中，配置bind 主机ip，在从redis服务器redis.conf中，配置bind 从机ip，并在下面加入slaveof 主机ip 端口，然后重启两台redis服务
